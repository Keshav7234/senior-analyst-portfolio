# Senior Analyst Portfolio

A stunning, modern portfolio website for a Senior Analyst at HCL Technologies.

## Features

✨ **Modern Design**
- Beautiful gradient backgrounds
- Smooth animations and transitions
- Responsive layout
- Dark theme with neon accents

🎯 **Sections**
- Hero section with call-to-action buttons
- About section with skills and achievements
- Expertise showcase with color-coded cards
- Featured projects with gradient headers
- Contact information and form
- Responsive footer

🚀 **Technologies**
- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox)
- JavaScript (ES6+)
- Responsive Design
- CSS Animations

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Keshav7234/senior-analyst-portfolio.git
```

2. Navigate to the project directory:
```bash
cd senior-analyst-portfolio
```

3. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
```

## Customization

### Update Personal Information

Edit the following in `index.html`:
- Contact email
- Phone number
- Location
- Project descriptions
- Social media links

### Modify Colors

Edit CSS variables in `styles.css`:
```css
:root {
    --primary: #1e3c72;
    --secondary: #2a5298;
    --accent: #00d4ff;
    /* ... more variables */
}
```

### Add More Projects

Duplicate a project card in the projects section and update the content.
